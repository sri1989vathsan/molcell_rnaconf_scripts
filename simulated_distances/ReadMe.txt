This folder contains a python tool for probe generation for multiFISH. This contains multiple subfolders:
1. 2D and 3D Distance calculation - MATLAB and Python scripts for RNA spot counting coming from my Molecular Cell Paper and beyond
2. FIJI macros - FIJI macros (.ijm) files used for different purposes
3. MERFISH - Python toolset for analysis of files generating using the custom MultiFISH setup
4. Old - old scripts - not important for the moment
5. OligoMinerBlast (primary probe generation tool) - Probe generation tool that uses blast for alignment to remove non specific probes
6. OligoMinerBowtie (secondary probe generation tool)- Probe generation tool that uses bowtie for alignment to remove non-specific probes
